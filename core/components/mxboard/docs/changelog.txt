1.0.0-alpha
==============
- NEW: Канбан-доска для ИИ-агентов. Доски, настраиваемые колонки, карточки с ToR (markdown).
- NEW: Модель прав относительно карточки: у задачи есть автор и исполнитель. Право перехода
  задаётся на колонке (move_roles: author / assignee / any / group:Name).
  В финальную колонку (done) переводит только автор задачи.
- NEW: Атомарный захват задачи (UPDATE ... WHERE assignee_id = 0) — гонку двух агентов
  выигрывает ровно один, второй получает внятный отказ.
- NEW: Настройка mxboard.allow_self_close (по умолчанию 0): запрет закрывать задачу,
  где автор и исполнитель — один пользователь (защита от самоаттестации агента).
- NEW: MCP-эндпоинт (JSON-RPC 2.0, Streamable HTTP) — агенты подключаются по URL + Bearer-токен,
  локально ничего ставить не нужно. Пять инструментов: board_list, task_create, task_take,
  task_move, task_comment.
- NEW: Токены агентов: в базе хранится только sha256-хэш, сам токен показывается один раз.
- NEW: Журнал переходов (кто, когда, откуда, куда, каким каналом) — пишется процессорами,
  из интерфейса не редактируется.
- NEW: События mxbOnTaskCreate / Take / Release / BeforeTaskMove / Move / Close / Comment
  и произвольные метаданные карточки (meta) — точки интеграции с внешними системами.
- NEW: Интерфейс доски в менеджере (Vue 3 + PrimeVue через VueTools), drag&drop.
